# Swan vs passive porn blockers

Swan is not trying to be the broadest porn blocker. It is built around one moment: opening a site you already chose to track.

At that moment, Swan redirects the tab and calls you. The call is the difference.

## Passive blockers

Passive blockers are useful when the goal is to make sites harder to reach. Many focus on block lists, DNS filtering, device controls, or accountability reports.

Swan does less at the network and device level. It focuses on the browser moment where the urge becomes action.

## Accountability tools

Accountability tools often make behavior visible to someone else. That can help some people, but it also adds a social and privacy tradeoff.

Swan Core keeps the loop local by default. It does not send reports to another person or to a Swan-operated server.

## Swan's tradeoff

Swan is strongest when you want immediate interruption, not broad monitoring. It is weaker if you need device-wide blocking, image classification, operating-system controls, or managed accountability.

If that tradeoff matches what you need, start with [Start](./).
